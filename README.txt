# A data challenge platform

A simple data challenge platform, motivated by educational purposes, in the context of a Machine Learning course. The code can be run on a server accepting python (Django) wsgi.

### The main tools used 

* Web framework: Django 
* Data results processing: sklearn + pandas
* Styling library: Bootstrap
* Database: SQLite

## Installation of the development environment

### On linux platforms
  Run the command `install.sh`

Note: Using a virtual environment for python is recommended
